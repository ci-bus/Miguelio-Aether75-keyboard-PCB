# Miguelio Aether75 keyboard PCB

Mechanical keyboard 75% Bluetooth

[Sales available soon at:](https://www.miguelio.com) 

### Main features

- 75% ANSI + ISO
- Hotswap mx sockets
- ZMK firmware configurable with ZMK Studio
- JST USB connector
- Encoder
- RGB Underglow LEDs
